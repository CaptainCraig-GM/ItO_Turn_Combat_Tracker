# ItO_Turn_Combat_Tracker
a simple python tkinter script that handles some book keeping while running Onto the Odd style games

Keeps an internal clock, tracks torches/timers
Little lists of random names and items to find

Toggle to a Combat Tracker that simplifies tracking HP on enemies (if you set it up before hand :P )
